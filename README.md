# Rick and Morty Archive

## Installation
Install dependencies.
```
npm install
```

### Development mode
Start development mode.
```
npm run dev
```


### Production mode
1. Build assets.
```
npm run build
```
2. Start production mode.
```
npm start
```

## Testing

Run unit and integration tests with jest.
```
npm test
npm run test:watch
```

Run end to end tests with cypress.
```
npm run cy:open
```
